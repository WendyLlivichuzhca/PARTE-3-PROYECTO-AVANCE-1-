# EduCampus — Sistema de Cursos Online 🎓

EduCampus es una plataforma integral de gestión educativa desarrollada en **Python** utilizando **CustomTkinter** para una interfaz moderna y **SQLite** para la persistencia de datos. El sistema sigue una arquitectura modular (MVC) y cumple con requerimientos de alta escalabilidad.

## 🚀 Funcionalidades Principales

-   **Dashboard dinámico**: Estadísticas en tiempo real de ingresos y actividad.
-   **Gestión Integral**: Administración de instructores, estudiantes y cursos.
-   **Modo Examen**: Evaluaciones interactivas con calificación automática.
-   **Certificación**: Generación automática de certificados en PDF (100% progreso).
-   **Recursos y Materiales**: Biblioteca de enlaces y documentos descargables.
-   **Foro Académico**: Canal de comunicación entre alumnos y docentes.

## 🛠️ Tecnologías

-   **Frontend**: CustomTkinter / Tkinter
-   **Backend**: Python 3.x
-   **Base de Datos**: SQLite3
-   **Reportes**: ReportLab (Generación de PDF)

## 📂 Estructura del Proyecto

-   `/controllers`: Lógica de comunicación entre vista y servicio.
-   `/database`: Esquemas y configuración de conexión.
-   `/models`: Entidades de negocio.
-   `/services`: Lógica de negocio y reglas del sistema.
-   `/views`: Interfaz gráfica y diseño de usuario.
-   `/utils`: Utilidades (PDF, i18n, etc.).
